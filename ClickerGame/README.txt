= Introdução =

O jogo chama-se Ru Clicker e conta da perspectiva dos administradores do
Restaurante Universitário sobre o aumento do valor dos passes para os 
estudantes. Sua jogabilidade baseia-se em clicar na bandeja para comprar 
coisas que ajudam os administradores a aumentarem o valor dos passes, além
de upgrades que são desbloqueados conforme a evolução do jogo.

= Como executar =
Para rodar, execute Debug/ClickerGame.exe no Windows
ClickerGame.sln é a solução do VisualStudio 2017
